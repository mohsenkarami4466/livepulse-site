# راهنمای خیلی ساده Deploy

## `npm run deploy` چیست؟

این یک دستور ساده است که:
1. کد شما را build می‌کند (تبدیل به فایل‌های آماده)
2. فایل‌ها را به فولدر `docs` کپی می‌کند
3. آماده می‌کند برای push به GitHub

---

## مراحل ساده (برای هر بار تغییر):

### مرحله 1: Build و آماده کردن
```bash
npm run deploy
```

این دستور خودکار:
- Build می‌کند
- فایل‌ها را به `docs` کپی می‌کند
- آماده می‌کند

### مرحله 2: Commit و Push
```bash
git add docs/
git commit -m "deploy"
git push origin main
```

---

## خلاصه:

**هر بار که کد را تغییر دادید:**

```bash
npm run deploy
git add docs/
git commit -m "deploy"
git push origin main
```

**همین!** بعد از 1-2 دقیقه، تغییرات در سایت نمایش داده می‌شود.

---

## اگر `npm run deploy` کار نکرد:

می‌توانید به صورت دستی انجام دهید:

```bash
npm run build
rm -rf docs/*
cp -r dist/* docs/
cp .nojekyll docs/
git add docs/
git commit -m "deploy"
git push origin main
```

---

## نکته:

- تنظیمات GitHub Pages باید روی **branch: main** و **folder: /docs** باشد
- بعد از push، 1-2 دقیقه صبر کنید تا سایت به‌روز شود
